Network Access Form

Overview

The purpose of this form is to help you obtain network/computer access for a new person starting work at OCC. If the person does not need any type of computer access, this form is not necessary.

If the person requires computer access, then ALL FIELDS, (except the gray areas) including date of birth and address, are required – so that Information Technologies (IT) can assign a network password. The personal information (date of birth and address) is required so that IT can validate the person’s identity if they should need their password reset.
